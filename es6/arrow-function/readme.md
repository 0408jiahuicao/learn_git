## 怎么用箭头函数
- 只有一个参数可以省略括号
- 函数只有一条语句可以省略 {} return
返回对象时候要加 () 

## arguments
- 所有函数中(除了箭头函数)都可用的局部变量
- [arg1, arg2, arg3]

## 普通函数与箭头函数的区别
- 箭头函数不支持重名形参，普通函数可以
- 箭头函数没有 this，他的this指向定义时所处的上下文(外部函数) this
- 箭头函数不能通过 .call .apply ... 改变箭头函数的this
- 箭头函数没有 .prototype
- 箭头函数不能作为构造函数
- 箭头函数没有arguments
- 箭头函数 没有 new.target
